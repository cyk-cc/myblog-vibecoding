# 2026-04-18 工作记录

## 梳理 myblog 项目结构
- 项目为基于 Flask 的轻量级本地博客，采用 Markdown 文件驱动 + SQLite 持久化设计
- 四层架构：app.py（入口）→ view.py（路由）→ service.py（业务）→ models.py（数据）

## 改动：访问统计功能
- **models.py**：posts 表新增 `views INTEGER DEFAULT 0` 字段，自动 ALTER TABLE 迁移旧数据库；新增 `Post.increment_views()` 和 `Post.get_site_stats()` 方法
- **view.py**：文章详情页访问时自动调用 `increment_views`，新增 `/stats` 路由展示总览+热门文章排行
- **templates/index.html & post.html**：meta 区新增 `👁 xx 阅读` 标签
- **templates/stats.html**：新建统计页，含总文章数/总阅读量/篇均阅读三张卡片 + 热门文章 Top10 排行表
- **base.html**：导航栏新增"访问统计"入口
- **style.css**：补充 `.views-count`、`.stats-page`、`.stat-card`、`.stats-table` 等样式

- **service.py**：调整日期提取顺序，改为文件名日期 > 文件内 `date:` 元数据 > 当天日期（兜底）
- **view.py**：新增 `/upload` 路由，上传 .md 文件后自动按 `上传日期-原文件名.md` 格式重命名保存，并立即同步到数据库
- **templates/upload.html**：新建上传页面，支持点击选择和拖拽上传
- **templates/base.html**：导航栏加入"上传文章"入口
- **static/style.css**：补充上传页面样式
- **app.py**：添加 `SECRET_KEY` 以支持 flash 消息
