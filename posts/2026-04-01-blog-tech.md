# 本地博客系统技术说明

## 项目概述

这是一个基于 Flask 的轻量级本地博客系统，采用 **vibe coding** 方式开发。无需复杂的数据库配置，纯静态 Markdown 文件即可驱动整个博客。

## 技术栈

| 组件 | 技术选择 | 说明 |
| :--- | :--- | :--- |
| 后端框架 | Flask | 轻量级 Python Web 框架 |
| 模板引擎 | Jinja2 | Flask 内置，支持模板继承 |
| 数据库 | SQLite | 无需配置，文件即数据库 |
| 文章格式 | Markdown | 支持代码高亮、目录等扩展 |
| 代码高亮 | highlight.js | 客户端自动高亮 |
| AI 辅助 | Kimi 2.5 (代码) + GPT-4o (Debug) | vibe coding 主力 |

## 目录结构

```
my_blog/
├── app.py                 # 主应用 (Flask)
├── posts/                 # 存放 Markdown 文章
│   └── 2026-03-31-hello.md
├── templates/             # Jinja2 模板
│   ├── base.html         # 基础模板
│   ├── index.html        # 首页模板
│   └── post.html         # 文章页模板
├── static/                # CSS, JS, 图片
│   └── style.css
└── blog.db                # SQLite 数据库文件
```

## 核心功能

### 1. 自动文章加载
- 启动时自动扫描 `posts/` 目录
- 解析所有 `.md` 文件并写入 SQLite 数据库
- 文件名格式：`YYYY-MM-DD-slug.md`（如 `2026-04-01-hello-world.md`）
- 修改 Markdown 文件后重启应用即可更新

### 2. Markdown 支持
- 标准 Markdown 语法
- 代码高亮（支持 Python、JavaScript 等多种语言）
- 目录生成（TOC）
- 表格、引用等扩展

### 3. 响应式设计
- 移动端适配
- 简洁美观的阅读体验

### 4. IPv6 支持
- 监听 `::` (IPv6 all interfaces)
- 端口：`5111`

## 启动方式

```bash
cd my_blog
python app.py
```

访问：`http://localhost:5111`

## 路由说明

| 路由 | 说明 |
| :--- | :--- |
| `/` | 首页，文章列表 |
| `/post/<slug>` | 文章详情页 |

## 如何发布新文章

1. 在 `posts/` 目录新建 Markdown 文件
2. 文件名格式：`YYYY-MM-DD-标题slug.md`
3. 第一行 `# 标题` 作为文章标题
4. 重启服务即可

## 开发过程

1. **框架生成**：使用 dsv3 生成基础结构
2. **代码生成**：Kimi 2.5 提供代码实现
3. **问题排查**：GPT-4o 协助解决 jinja2 模板错误

### 问题记录

**问题**：`jinja2.exceptions.UndefinedError: 'now' is undefined`

**原因**：在 `app.py` 中使用 `app.jinja_env.globals['now'] = datetime.now` 定义的全局变量在模板中调用方式不正确。

**解决**：改用 `context_processor` 注入当前年份：

```python
@app.context_processor
def inject_current_year():
    return dict(current_year=datetime.now().year)
```

模板中使用 `{{ current_year }}` 代替 `{{ now().year }}`。
